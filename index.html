SeaMonkey sidebar directory

 

This is a list of web pages that work as sidebar panels for SeaMonkey. Most of these pages will also work as Firefox sidebar bookmarks or Vivaldi web panels.

Key:
S This site was designed specifically to be used as a sidebar or web panel.
J Requires JavaScript.
F Requires Flash.
P Installs as a persistent panel in SeaMonkey. Closing the sidebar will close the site, but simply hiding the sidebar with the handle or switching to a different panel will not.

Last updated: June 10, 2017

Reference

LookWAYUp S
Provides a dictionary and translation between English and French Spanish, German, and Dutch.

HTML Quick Reference S
HTML element and attribute reference from r68.nl. Links to the spec on www.whatwg.org.

CSS Quick Reference S
CSS 2.1+ property and value reference from r68.nl. Links to the spec on www.whatwg.org.

JavaScript Guide S
Also hosted at r68.nl, an index for MDN's JavaScript Guide. Links to MDN open in a new tab.

Python Sidebar S
A Python documentation reference from Edgewall.

Unicode Charts S J
A Unicode character reference organized by block, from aleto.ch.

News and Information

NPR
The home page of U.S. radio network NPR. Hold Ctrl while clicking to open a link outside of the sidebar.

MLB Scoreboard
Scores of current and completed Major League Baseball games, from NBC Sports.

spacetoday.net S
Links to and summaries of space-related news.

Tools

Clockbar S F
A Flash-based analog clock (200x200 pixels) based on your computer's local time. From edmullen.net.

Stopwatch S J
A JavaScript stopwatch with a running total. Can display the time in seconds or tenths, hundredths, or thousandths of a second.

Web Applications

Google Calendar
The mobile version of Google Calendar. Allows you to view upcoming events and quickly add new ones.

Twitter
The mobile version of Twitter.

Skype J
The web version of Skype.
