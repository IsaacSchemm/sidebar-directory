SeaMonkey sidebar directory

Add and preview buttons will not work with JavaScript disabled.

This page is a list of sidebar panels for SeaMonkey. Most of these pages will also work as Firefox sidebar bookmarks or Vivaldi Web Panels.

Key:
J Requires JavaScript.
F Requires Flash.
P Installs as a persistent panel in SeaMonkey; closing the sidebar will log you out, but simply hiding the sidebar with the handle or switching to a different panel will not.

Last updated: June 10, 2017

Reference

LookWAYUp
Provides a dictionary and translation between English and French Spanish, German, and Dutch.

News and Information

spacetoday.net
Links to and summaries of space-related news.

Tools

Clockbar F
A Flash-based analog clock (200x200 pixels) based on your computer's local time.

Stopwatch J
A JavaScript stopwatch with a running total. Can display the time in seconds or tenths, hundredths, or thousandths of a second.

Chat and Social

Skype J
The web version of Skype.
